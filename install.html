How to install electron (The fast way)

There is a 'non-quickstart' way to do this, but since we need to do this quickly
  • # Clone the Quick Start repository
    $ git clone https://github.com/electron/electron-quick-start
  • # Go into the repository
    $ cd electron-quick-start
  • # Install the dependencies and run
    $ npm install && npm start

Instructions from electron.atom.io

Main.js

main.js is the file that has your electron config data. This is the first thing ran to get your app up and running.

You can also edit settings in main.js to make your app frameless, open up the developer console and much more.

Index.html

By default, the file that your html starts with is index.html. This is the front page to your app. You can add multiple pages from here and even create some form of navigation.
