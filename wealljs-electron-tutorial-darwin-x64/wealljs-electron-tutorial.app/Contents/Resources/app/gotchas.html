Gotcha's

If you are going to use Angular/React or another framework...

Import the Framework before you load electron.

jQuery needs to be imported and loaded

install jQuery through npm

npm install -g jQuery

Use the following in your html to get jQuery working

<script> try { $ = jQuery = module.exports; } catch(e) {} </script>
<script>window.$ = window.jQuery = require('jquery');</script>
<script src="index.js" />

Be Careful with </a> tags

Since the electron window is a browser window, it will load urls in the existing browser (even if you use target="_blank" or "_new"). To load urls into a browser, you can use the shell method.

const {shell} = require('electron')

// for your a tag <a href="#" onClick="shell.openExternal('http://wealljs.org')"">WeAlljs.org</a>"
